# Trading Signals: News Sentiment + Price Anomalies

3-day AI coursework project combining LSTM sentiment classification with VAE anomaly detection.

## Structure
- `data/`: Raw Kaggle datasets
- `notebooks/`: Jupyter notebooks for each phase
- `src/`: Modular Python code (data loading, models, signal generation)
- `results/`: Trained models and plots

## Setup
```bash
git clone <your-repo>
pip install -r requirements.txt
```

## Usage
1. Download Kaggle datasets to `data/`
2. Run notebooks in Colab
3. Import from `src/` modules
